Convert Closed Captions from ".srt" file to Transcripts


 Open the ".srt" file in Notepad, gedit or your favorite text editor and paste its contents into the field below. Then click on "Convert to a transcript" to watch the magic happen right before your eyes!


Note that this convertor will not work if the subtitle file you have is not an srt file (examples include dxfp, ssa, ttml files). If your subtitle file is not srt, please use this link to convert the file into an srt file.
